# GameHub - 本地游戏管理器

一款基于 PyQt6 的本地游戏管理与启动工具，提供深色主题界面、游戏分类浏览、时长统计、隐私模式等功能。

## 功能特性

- **游戏管理** - 手动添加或自动扫描目录导入游戏，支持编辑和删除
- **分类浏览** - 按类型分类（动作、策略、RPG、冒险等），侧边栏快速筛选，支持自定义分类
- **搜索** - 实时搜索游戏名称
- **一键启动** - 卡片悬浮点击或详情页启动游戏，实时显示运行状态
- **时长统计** - 自动记录每次游玩时间，累计总时长
- **封面管理** - 自动生成渐变封面，支持自定义封面、网络搜索封面并裁剪
- **隐私模式** - 游戏封面马赛克处理，名称遮蔽
- **详情页** - 展示游戏封面、简介、分类、游玩时长、启动参数等完整信息
- **深色主题** - 现代深蓝风格 UI，自定义 QSS 样式

## 安装与运行

```bash
pip install -r requirements.txt
python main.py
```

依赖：Python 3.10+，PyQt6 >= 6.5.0

## 使用说明

1. **添加游戏** - 点击顶部「+ 添加游戏」手动添加，或点击「扫描目录」自动扫描
2. **启动游戏** - 悬浮卡片点击播放按钮，或进入详情页点击启动
3. **浏览分类** - 左侧边栏切换分类，支持搜索
4. **分类管理** - 设置中可添加或删除分类，删除分类后该分类下的游戏自动归入「其他」
5. **隐私模式** - 设置中开启，封面马赛克、名称遮蔽
6. **自定义封面** - 详情页中搜索网络封面，或手动选择本地图片
